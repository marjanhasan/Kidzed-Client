# kidzed

## WELCOME TO Kidzed TOY STORE.

You will find the best classic, whimsical and educational toys for kids of all ages at kidzed Toy Store using MERN stack, Tailwind CSS & Firebase technology.

## Live Link

https://kidzed-bbaa3.web.app/

## Features

- MERN stack project, Tailwind CSS
- Mobile, Tablet & DeskTop Responsive
- Firebase Authentication(Google,Github,Email)
- React Router Dom
- AOS Packages
- Dynamic Title like GMAIL
- React Tabs
- Sweet Alert
- React ToolTip
- Add toys
- Update/Delete toys
- All toys from different user
